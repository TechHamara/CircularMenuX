## <kbd>Events:</kbd>
**CircularMenuX** has total 2 events.

### 💛 ItemClicked
Triggered when an item in the Circular Menu is clicked.

| Parameter | Type
| - | - |
| index | number

### 💛 CenterClicked
Triggered when the center item of the Circular Menu is clicked.
